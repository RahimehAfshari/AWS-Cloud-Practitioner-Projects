# AWS Security Labs

This repository documents my hands-on learning in AWS Cloud and Security.
The focus is on core cloud services, security best practices, and practical labs.

## Goals
- Build strong foundations in AWS cloud concepts
- Understand cloud security fundamentals
- Apply security best practices in real scenarios
- Document learning clearly and professionally

## Topics Covered
- AWS IAM (Users, Roles, Policies)
- S3 Security (Permissions, Public vs Private Access)
- EC2 Basics
- Cloud Monitoring (CloudWatch)
- Security Principles (Least Privilege, Access Control)

## Labs

### Lab 1: AWS IAM Basics
**Objective:**  
Understand IAM users, roles, and policies.

**What I practiced:**
- Creating IAM users
- Assigning permissions using policies
- Applying least privilege principles

**Key Learnings:**
- Importance of access control
- Difference between users and roles
- Security risks of over-permission
